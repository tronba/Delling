# Project Box — STL Files

STL files for the project enclosure. Versions are available for the **Orange Pi Zero 3** and **Raspberry Pi 4/5**.

There are two versions of the main box: with text and without.

> Printed on a Bambu Lab X1 Carbon. If using the text version, colour the inside of the text in Bambu Studio before slicing.

---

## Bill of Materials

### Orange Pi Zero 3

- Orange Pi Zero 3 (1.5 GB+)
- 8 GB+ Micro SD card
- SDR dongle (e.g. RTL-SDR Blog V4)
- USB memory stick 64 GB+
- Telescopic antenna 147–633 mm
- 40×40×10 mm 5 V fan
- 20 cm panel-mount USB-C Male to Female extension (main power)
- Antenna connector / cable from SDR to antenna mount

> The Orange Pi Zero 3 has only one USB-A port. Two extra ports are available via the pinout — a bought or homemade cable is required.
> See the [Orange Pi Zero 3 hardware page](http://www.orangepi.org/html/hardWare/computerAndMicrocontrollers/details/Orange-Pi-Zero-3.html) for pinout details.

**Additional cables for extra USB ports:**

- 6-pin 2.54 mm Dupont single-row header
- USB 2.0 Female 30 cm with four wires (data + power for SDR)
- USB-C Male 30 cm with four wires (data + power for Meshtastic)

---

### Raspberry Pi 4/5

- Raspberry Pi 4 or 5 (2 GB+)
- 8 GB+ Micro SD card
- USB memory stick 64 GB+ (50 mm max length)
- SDR dongle (e.g. RTL-SDR Blog V4)
- Telescopic antenna 147–633 mm
- 40×40×10 mm 5 V fan
- 20 cm panel-mount USB-C Male to Female angled extension (main power)
- 15 cm USB-A Male to Female extension (for SDR)
- Antenna connector / cable from SDR to antenna mount
